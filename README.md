# Mr. Foodie - Express Online Food Delivery Web Application

![Mr. Foodie Banner](https://images.unsplash.com/photo-1504674900247-0877df9cc836?auto=format&fit=crop&w=1200&q=80)

**Mr. Foodie** is a modern, premium, and fully responsive food delivery web application inspired by leading food tech platforms like Zomato. Built using clean HTML5, CSS3, Bootstrap 5, and Vanilla JavaScript, it offers a seamless food ordering experience tailored for authentic Indian cuisine and state-wise regional specialties.

---

## 🌟 Key Features

- **Redesigned Indian Culinary Theme**:
  - Authentic Indian restaurant brands (e.g., *Punjabi Tadka*, *Dosa Plaza*, *Hyderabadi Biryani House*, *Dilli Zaika*).
  - 20+ Regional Indian Dishes spanning Punjab, Tamil Nadu, Andhra Pradesh, Kerala, West Bengal, Maharashtra, Bihar, Gujarat, and Rajasthan.
  - Prices represented in Indian Rupees (`Rs.`).
  
- **Clean & Emoji-Free UI**:
  - Professional Font Awesome 6.5 & Bootstrap 1.11 icons across the header, categories, cards, and footer.
  - Curated color palette (Crimson Red `#E23744`, Pure White, Dark `#1C1C1C`, Light Gray).

- **Dynamic Interactive Functionality**:
  - **Live Search**: Instant real-time filtering across restaurants, cuisines, and dish names.
  - **Category & Dietary Filtering**: Filter by Pure Veg, Non-Veg, Cuisines, Ratings, and Sort by price or popularity.
  - **Quick View Modal**: Inspect dish details, prep time, and ingredient list before ordering.
  - **Interactive Offcanvas Cart**: Dynamic subtotal, GST (5%), delivery fees (Free above Rs.299), quantity controls, and simulated checkout flow with toast notifications.
  - **Sticky Navbar & Smooth Scrolling**: Responsive layout optimized for Mobile, Tablet, and Desktop.

---

## 🛠️ Built With

- **HTML5**: Semantic document layout & SEO tags
- **CSS3**: Custom CSS tokens, glassmorphism, smooth animations, and responsive utilities
- **Bootstrap 5.3**: Grid layout, carousels, offcanvas drawer, and modal components
- **Vanilla JavaScript (ES6+)**: Modular application architecture without external frameworks
- **Font Awesome & Bootstrap Icons**: Vector icons for crisp rendering
